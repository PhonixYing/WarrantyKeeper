# Warranty Keeper - Official Website (GitHub Pages)

This folder is a static website designed for GitHub Pages deployment.

Current repo:
- `https://github.com/PhonixYing/WarrantyKeeper`
- Recommended public site URL: `https://phonixying.github.io/WarrantyKeeper/en/`

## Structure

- `index.html` - Redirects to English site
- `en/index.html` - Main landing page (English)
- `privacy.html` - Privacy Policy (public URL for App Store)
- `terms.html` - Terms of Use (public URL for App Store)
- `assets/` - Icons + placeholder screenshots (replace with real assets later)

## Deploy with GitHub Pages

1. Create a new GitHub repo (or use an existing one).
2. Copy this folder's contents to the repo root.
3. In GitHub: Settings -> Pages -> Deploy from a branch -> select `main` (or `master`) and `/ (root)`.
4. Your site will be available at:
   - `https://<username>.github.io/<repo>/en/`
